# Introduction
Implements the [AfriGIS Geocoding (Search) Service](https://developers.afrigis.co.za/portfolio/search/)
# Building
`mvn clean package`

# Use as dependency
```xml
<dependency>
	<groupId>com.afrigis.services</groupId>
	<artifactId>geocode</artifactId>
	<version>0.0.16</version>
</dependency>
```
